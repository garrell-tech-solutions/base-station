# OpenClaw Base Station - Backup Content UI

This repository stores backups of learning content (Florida law, Broward, local) and provides a minimal UI to browse them via GitHub Pages.

## Repository Layout
- `florida-law-learning/` - Florida law learning content
- `broward-learning/` - Broward County learning content
- `local-learning/` - Local / Coral Springs learning content
- `docs/` - Static UI assets served via GitHub Pages

## Hosting
GitHub Pages, Source: main branch /docs

## Adding Content
1. Place new files under the appropriate learning folder
2. Commit and push to main
3. GitHub Pages reflects updates automatically
